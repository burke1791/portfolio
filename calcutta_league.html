March Madness Calcutta Tournament

Status: Work In Progress

Description

A fully functional site that can host your March Madness Calcutta tournament from hosting the live auction at the start to calculating each user’s payout at the end. This app was built using ReactJS with Hooks, Reach Router, and Ant Design Styled Components for the front end, and MySQL combined with Google’s Firebase for the back end.

The front end utilizes a publisher subscriber data pattern to keep each component up to date with info from the databases. Using the Firebase Authentication API, the app will only display the tournaments that the user has created or joined, resulting in a personalized user experience.

Development is ongoing with many new features on the way, including more options to socially interact with your friends and a number of data visualization options for the auction results and your teams' performance.

Motivation

I created this app because I couldn't find another site out there that kept track of a Calcutta tournament from start to finish. Before I created this website, we were stuck doing a live auction via Skype and then keeping track of everything in a Google Spreadsheet.

Now that we use this app, everything about the tournament experience is much smoother. The auction can handle any number of simultaneous bids without error, and back end cloud functions keep the dollar values up to date as the games end.
