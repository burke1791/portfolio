Big Ten Hangman - Browser Game

Status: Complete

Description

This is a simple hangman style browser game where the options are Big Ten schools. You start the game by pressing the spacebar and are only allowed two incorrect guesses. If you successfully guess the school, its logo will appear below the game area.

This app was built with basic HTML and styled with the Bootstrap grid system, while jQuery powers the DOM manipulation.
