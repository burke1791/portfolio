Flockatime

Status: Work In Progress

Description

This single page app is a high level clone of the popular WakaTime productivity tool. The front end is built with ReactJS v16.8 and Bootstrap, while Googleâ€™s Firebase powers the back end - both authentication and database management. The use of this app requires installation of the Visual Studio Code extension of the same name - Flockatime. As you work in VS Code, the extension periodically sends timestamps attached to your unique identifier up the the NoSQL database, which the front end computes into human readable data.

Once you sign in, the app displays a bar chart visually representing the amount of time you spent coding over the past seven days, as well as a global leaderboard that shows every user with an account.

Future work will focus on diving deeper into everything React Hooks has to offer, as well as smoothing out the user experience.

Motivation

My motivation for this project was simple: Learn about React Hooks. I built the original Flockatime website with several friends using standard HTML, CSS, and Javascript and felt that the scope of this app was small enough that I wouldn't be overwhelmed trying to build it with the newest version of React. I am still learning everything that Hooks has to offer, but the experience has been very enlightening thus far.
