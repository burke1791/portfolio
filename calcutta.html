College Basketball Calcutta Tournament

Status: Complete - v0

Description

A fully functional site that can host your March Madness Calcutta tournament from hosting the live auction at the start to calculating each user’s payout at the end. This app was built using ReactJS (before the introduction of Hooks), React Router, and Bootstrap for the front end, and Google’s Firebase for the back end.

The front end utilizes a publisher subscriber data pattern to keep each component up to date with info from the NoSQL database. Using the Firebase Authentication API, the app will only display the tournaments that the user has created or joined, resulting in a personalized user experience.

In the future, I plan to rewrite this app from scratch using React v16.8+ to take advantage of the new Hooks feature, as well as incorporate the Context API for state management. I will also look to improve the UI/UX and functionality as a whole with everything I’ve learned since completing “version zero.”
